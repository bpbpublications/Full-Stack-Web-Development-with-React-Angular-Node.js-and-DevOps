# Chapter 02 — Styling HTML with CSS (Code Bundle)

This bundle contains Chapter 2 examples and a hands-on project.

## How to run (quick)
- Open any `index.html` file in a browser.
- For examples that use a CSS file, keep the folder structure intact.
- Hands-on project entry point:
  `project_styled_responsive_page/index.html`

## Folder map
- `examples/`  → topic-wise examples (each has its own mini folder)
- `assets/`    → sample image used in backgrounds/cards
- `project_styled_responsive_page/` → styled + responsive multi-page website

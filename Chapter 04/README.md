# Chapter 04 — JavaScript Fundamentals (Code Bundle)

This bundle contains example code files for core JavaScript concepts and a mini project.

## How to run (browser)
1. Open any example `index.html` file in a browser.
2. Press **F12** (DevTools) → open the **Console** tab to see output.

## How to run (Node.js) — optional
If Node.js is installed, run some scripts directly:
- `node examples/xx_folder/script.js`
- `node project_number_guess_game/node_game.js`

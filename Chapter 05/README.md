# Chapter 05 — JavaScript and DOM Manipulation (Code Bundle)

This bundle contains DOM examples and a hands-on project.

## How to run
- Open any `index.html` file in a browser.
- Use DevTools → Console to see logs.

## Topics covered (examples)
- DOM selection (getElementById, querySelector, querySelectorAll)
- Events (click, input, keydown, submit)
- Creating and removing elements
- Class and style manipulation
- Form handling and validation
- LocalStorage basics

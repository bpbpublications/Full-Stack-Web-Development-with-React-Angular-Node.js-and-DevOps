Chapter 7 – React (Todo App Project Bundle)

This folder contains the main React Todo application code shown in Chapter 7:
- A reusable useLocalStorage hook
- TodoForm, TodoItem, and TodoList components
- App component that wires everything together
- Basic App.css styling and index.js entry point

You can drop the contents of 08_todo_project into a fresh Create React App or Vite + React
project (adjusting paths if needed) to run the example.

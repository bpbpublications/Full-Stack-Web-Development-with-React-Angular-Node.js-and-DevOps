# Chapter 01 — HTML (Code Bundle)

This folder contains all Chapter 1 code files (examples + hands-on project).

## How to run
- Open any `.html` file in a web browser (Chrome/Edge/Firefox).
- For the mini project, open:
  `project_simple_static_website/index.html`

## Folder map
- `examples/`  → chapter-wise example code files
- `assets/`    → sample media used by examples (image/video/pdf)
- `project_simple_static_website/` → hands-on project (index/about/contact + styles.css)

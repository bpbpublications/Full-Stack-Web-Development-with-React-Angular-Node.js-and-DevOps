# Assets

This folder contains small sample media files so that Chapter 1 examples work immediately.

- `image.jpg`  → used by the `<img>` example
- `poster.jpg` → used by the `<video poster="...">` example
- `video.mp4`, `video.ogg` → used by the `<video>` example
- `document.pdf` → used by the download link + `<iframe>` example

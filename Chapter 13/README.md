Chapter 13 – DevOps for Web Development (Code Bundle)

This bundle collects complete, runnable examples that correspond to the DevOps
topics in Chapter 13: Docker, Docker Compose, Kubernetes, CI/CD with GitHub
Actions, logging with Winston, and metrics with Prometheus + Express.

Folders:

01_docker_node_app/        – Node.js app, Dockerfile, and docker run commands
02_docker_compose/         – docker-compose.yml with web + Postgres db
03_kubernetes_manifests/   – Deployment and Service YAML for a web app
04_github_actions_ci/      – Sample GitHub Actions workflow for Node build/test
05_logging_winston/        – Winston logger configuration and usage
06_metrics_prometheus/     – prom-client integration and /metrics endpoint
99_raw_snippets/           – chapter13_all_code_snippets_raw.txt (all code-like
                             snippets extracted from the PDF so no example is lost)

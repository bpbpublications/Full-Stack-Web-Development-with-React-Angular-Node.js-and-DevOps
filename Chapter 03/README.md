# Chapter 03 — Advanced CSS Concepts (Code Bundle)

This bundle includes advanced CSS examples and a mini project (modern landing page).

## How to run
- Open any example `index.html` in a browser.
- Mini project entry point:
  `project_modern_landing_page/index.html`

## Topics covered (examples)
- CSS variables + theming
- Positioning, z-index, sticky
- Overflow, object-fit
- Transforms, transitions, animations
- Gradients, shadows, glass effect
- Advanced layout: Grid + Flex patterns
- Responsive type and spacing (clamp, calc)

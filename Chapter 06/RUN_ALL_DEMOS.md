# Run all Chapter 6 demos (quick commands)

```bash
npm install

# 1) HTTP basics
npm run start:http

# 2) Request–response flow
npm run start:flow

# 3) HTTPS (after generating cert)
npm run start:https

# 4) Cookies + Sessions
npm run start:cookies
npm run start:sessions

# 5) Headers
npm run start:headers

# 6) CORS (run both)
npm run start:cors:api
npm run start:cors:web

# 7) REST API mini-project
npm run start:rest
```

React Todo List Project (Chapter 9)

This example follows the React hands-on project from Chapter 9.
It uses Create React App style structure (`src/` folder).

Quick start (with an existing CRA or Vite+React project):
1. Create a React project:
   npx create-react-app todo-app
   cd todo-app

2. Replace the contents of your src/ folder with the files in react_todo_app/src/
   (or merge them carefully).

3. Run the development server:
   npm start

Main features:
- Add, complete, and delete tasks.
- Separate components for App, TodoForm, TodoList, and TodoItem.
- Clean CSS styling for layout.

Chapter 9 – Hands-on Projects and Frameworks Comparison (Code Bundle)

This bundle gathers the main code used in Chapter 9:

1. React Todo List Project
   - Complete React SPA using functional components and hooks.
   - Covers component structure, props, state with useState, event handling,
     conditional rendering, and basic styling.

2. Angular User Management System
   - Angular CLI style structure with feature components and a service.
   - Demonstrates modules, components, services with HttpClient, routing,
     template-driven + reactive forms, and basic CRUD against a mock API.

3. Comparison Snippets
   - Small React and Angular examples that the chapter uses when comparing
     frameworks (useState usage, simple Welcome component, OnPush strategy, etc.).

Usage notes are included in each project’s local README.

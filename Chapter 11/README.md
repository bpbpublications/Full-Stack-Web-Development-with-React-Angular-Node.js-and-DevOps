Chapter 11 – Databases and Integration (Code Bundle)

This bundle collects all main code examples for the "Databases and Integration" chapter.
It includes runnable Node.js/Express + MongoDB (Mongoose) and PostgreSQL (Sequelize)
examples, plus raw snippets extracted from the book so that no code example is missing.

Folders:

01_sql_examples/            – SQL table definitions and basic queries
02_mongodb_mongoose/        – MongoDB connection, Mongoose schemas, CRUD routes
03_postgres_sequelize/      – Sequelize setup, models, associations, CRUD
04_indexing_and_queries/    – Index creation examples (Mongo + PostgreSQL)
05_transactions/            – Sequelize transaction example
06_student_course_project/  – Student Course Registration API (Express + Sequelize)
99_raw_snippets/            – chapter11_all_code_snippets_raw.txt (verbatim-ish extract)

Usage notes are provided in each folder's README and file comments.
